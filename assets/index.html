# 楼盘名片(1.3, 1.4修改后)【功能说明】

分享：

  1. 置业顾问分享 必带自己id
  2. 客户分享: 可能带置业顾问id
  3. 客户若无置业顾问分享为空分享
  4. 客户空分享：带空置业顾问id 00000000-0000-0000-0000-000000000000
    正常进入，没有授权置业顾问
    从没带置业顾问信息的分享进入
    服务器没有置业顾问信息，所有的分享

# 首页

进入方式

  1. 正常进入
  2. 分享进入{ 小程序分享 太阳短码 }

置业顾问：显示自身信息+互动列表【 上拉加载更多信息 下拉刷新页面 】
客户：根据授权信息显示 授权 或 置业顾问信息【 下拉刷新页面 】
顾客分享进入，根据进入的置业顾问，加载当前其信息 或 授权

# 楼盘页

进入方式

  1. 正常进入：查看楼盘信息
  2. 模板消息进入：替换置业顾问，查看楼盘信息

功能

  • 下拉刷新页面
  • 分享图canvas 绘制
  • 轮播图
  • 楼盘信息
  • 销售信息
    • 基础：
      • 楼盘表格：只在即将开盘下显示
      • 公示时间：只在销售公示和正在登记的状态下显示
      • 楼盘信息：销售公示期及以后都显示
      • 公示方案：同上
      • 房源价格：同上
      • 登记影响：同上
    • 客户：即将开盘下，具有开盘提醒的功能
  • 楼盘快讯
  • 周边配套：【显示当前大项的前3个，默认显示第一个，点击标明当前的点】

# 二级页面

  1. 朋友圈海报页
    置业顾问：进入无限制
    客户：具备置业顾问 且置业顾问的具有 PosterUrl 图片信息
  2. 楼盘相册页【点击进来，数据渲染点击的图片，上下两个列表对应显示】
  3. H5页面
  4. 房源价格页
  5. 意向登记页
  6. 楼盘快讯页
    1. 列表展示10条快讯，上拉加载更多
    2. 同一时间合并显示
  7. 户型列表页--> 户型详情页
    1. 列表显示当前楼盘户型
    2. 可根据楼盘 户型进行筛选
  8. 户型详情页 --> 普通用户可进入：聊天页
    • 进入方式
      • 正常进入:展示当前户型，户型列表
      • 分享进入: 基础功能:替换置业顾问，展示当前户型，户型列表
        • 顾客分享进入: 添加home 按钮、置业顾问信息、取消电话等按钮
    • 分享:分享当前户型信息，参数：置业顾问id,户型名称(唯一值)
    • 功能:
    1. 展示户型信息
    2. 绘制分享canvas图片 3、绘制海报
      • 置业顾问：自身信息(必有)
      • 顾客：根据是否授权区分能否分享
  9. 周边配套页
    1. 周边信息从前一级页面传递过来
    2. 显示所有信息
  10. 普通用户可进入：聊天页

  2. 动态详情（富文本解析）-->普通用户可进入：聊天页
